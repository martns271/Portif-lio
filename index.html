HABILIDADES

Html e Css

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para estruturar o conteúdo da web, enquanto CSS (Cascading Style Sheets) é a linguagem usada para estilizar esse conteúdo, definindo cores, fontes e layout.

JavaScript

JavaScript é uma linguagem de programação que permite criar interatividade e dinamicidade em páginas web, possibilitando manipular elementos HTML e CSS em tempo real.

C#

C# é uma linguagem de programação orientada a objetos desenvolvida pela Microsoft, amplamente utilizada para criar aplicativos em diversas plataformas, incluindo web, desktop e jogos.

Excel

Excel é um software de planilha da Microsoft que permite a manipulação, análise e visualização de dados por meio de fórmulas, gráficos e tabelas dinâmicas.
